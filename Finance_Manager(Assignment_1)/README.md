#Finance Manager
A financial manager website that has: - Dashboard
                                      - Income/Expense Form
                                      - Budgetting page 
                                      - Saving Goals adding page 
                                      - Transactions page showing your transactions 
                                      - Reports page with charts and graphs for visualy representing any data of your choice
